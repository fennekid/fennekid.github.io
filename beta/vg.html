Video Game Category Commands

Beta 42 has some commands that are specific to a particular game, though may have some use outside of that game.

Splatoon Commands

Splatoon is a Wii U game from Nintendo. It is a third person map-control shooter with a unique mechanic that you can turn into a squid and swim inside the ink you put on the map.

=schedule [2/60s]

"makes me pull a copy of the splatoon schedule"

Gets the current and next 2 cycles from the splatoon.ink API.

Usage:
=schedule ['format']
Gets the schedule. Format determines how the data is displayed.
mobile posts the data formatted for mobiles, classic posts the data in plain text.

=marie

"best squid"

=basket

"posts squids"

The squid basket was created by @Frawst and @LordToby

OneShot

OneShot is a PC game from Little Cat Feet and published by Degisa. It is an indie meta-adventure game that has some unique gameplay mechanics and solutions.

=textbox [2/5s]

This command posts an embed in the style of a OneShot textbox. Even though this command is labled as OneShot, it can work for a lot of other games too.
This command accepts any custom emoji.

If the bot has Manage Messages, it will delete the call message.

Usage:
=textbox 'character' 'content'
Creates a textbox. Character is who to use, content is what to say.

Creates a textbox with a custom character. Emote is a custom emoji, content is what to say

=solstice [1/30s]

"a styalised OneShot countdown"

This command uses your profile birthday and timezone and returns a countdown aimed at your birthday.
If you do not have a birthday, the timer will be turned off completely.
If you do not have a timezone, it defaults to UTC.

Usage:
=solstice ['member']
Member is who's birthday to count to. If not given, then it uses your own.

=entify [1/30s]

"creates a reflection of an image"

This command applies a purple scanline filter to whatever image was last sent into the channel.

Usage:
=entify
Fetches the last sent image and posts it.

Pokémon

Pokémon is a Nintendo franchise over several console generations created by Game Freak. It is a game about capturing small animals and forcing them to fight~!

=dex [2/25s]

"makes me search the Pokédex"

This searches the Pokédex and fetches a bunch of data about a given Pokémon.

Usage:
=dex 'number'
Number is the national dex number of the Pokémon to find.
=dex 'name'
Name is the (english) name of the Pokémon to find.

=pokestats [2/25s]

"makes me calcuate the maximum and minimum stats of a pokemon"

This searches the Pokédex and calculates the maximum and minimum stats of a Pokémon at levels 50 and 100.

Usage:
=pokestats 'name|num'
Name is the (english) name and num is the national dex number. Only one is required.

=movedex [2/25s]

"makes me get data about a Pokémon move"

This searches for a move and returns the data and statistics for it.

Usage:
=movedex 'name'
Name is the (quoted) name of the move to search for.

=abilitydex [2/25s]

"searches for a Pokémon ability"

This searches for a ability and returns its description and effects.

Usage:
=abilitydex 'name'
Name is the (quoted) name of the ability to search for.

=matchup [2/25s]

"makes me get Pokémon type matchup data on a type"

This searches for a Pokémon type and returns how it interacts when battling other types.

Usage:
=matchup 'type'
Type is the name of the type to search for.
